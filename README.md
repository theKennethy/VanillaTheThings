# VanillaTheThings

A comprehensive collection tracker addon for World of Warcraft 1.12.1 (Vanilla). Track your mounts, pets, recipes, reputations, exploration progress, and more!

## Features

- **Mount Collection** - Track all mounts with collected/missing status
- **Pet Collection** - Track all companion pets
- **Recipe Tracking** - Monitor tradeskill recipes across all professions
- **Reputation Progress** - Track faction standings
- **Exploration Tracking** - Monitor discovered areas and flight paths
- **Loot Database** - Comprehensive item database with source information
- **Tree Browser** - Navigate items by zone, instance, and source
- **Advanced Search** - Filter items by type, quality, source, and more

## Installation

1. Download or clone this repository
2. Copy the `VanillaTheThings` folder to your `Interface/AddOns/` directory
3. Restart WoW or type `/reload`

## Commands

| Command | Description |
|---------|-------------|
| `/vtt` | Toggle the tracker window |
| `/vtt mounts` | Open mount collection |
| `/vtt pets` | Open pet collection |
| `/vtt tree` | Open tree browser |
| `/vtt search <text>` | Search for items |
| `/vtt filter` | Open advanced search |
| `/vtt settings` | Open settings |
| `/vtt stats` | Show statistics |
| `/vtt refresh` | Refresh data |
| `/vtt help` | Show all commands |

Alternative command: `/vanillathethings`

## Requirements

- World of Warcraft 1.12.1 (Vanilla)
- Interface version: 11200
